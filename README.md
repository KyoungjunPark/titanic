Titanic project

> 2015 소프트웨어 공학 팀 프로젝트를 위한 repo
> 이 프로젝트의 목적은 관리/소프트웨어 설계를 개선할 수 있도록 소프트웨어 설계자가 소프트웨어 모듈의 전체 의존성을 이해하도록 돕는 DSM 툴을 개발하는 것이다.
기본적인 기능은 편집과 DSM의 시각화를 포함한다.
> 타이탄과 같은 현존하는 유사한 제품을 참조하며, 타이타닉 DSM은 타이탄과 호환이 가능해야 한다.
